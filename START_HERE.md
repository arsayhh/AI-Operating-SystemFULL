## What You Are Installing

You are installing a structured AI operating system.

Its purpose is to make ChatGPT:

- less generic
- less validating
- more directional
- more strict
- more action-oriented

This system is designed to:

- clarify your real problem
- filter weak signals
- eliminate bad options
- force a clearer next move

---

## Before You Begin

You need:

- access to ChatGPT
- access to ChatGPT settings
- the ability to open files in this repository one by one
- patience to follow the order

If you skip the order, the system may work poorly.

---

## Installation Principle

This system is not loaded all at once.

It is installed step by step.

Some parts belong in:
- ChatGPT settings
- memory / persistent preference context
- a new chat as runtime prompt

Each file has a role.
Use each file only for its intended purpose.

---

## Installation Order

Follow this exact order:

### Step 1
Open:

`SYSTEM_MANIFEST.md`

Read it first.
This file tells you what exists in the system and what is required.

---

### Step 2
Open:

`INSTALLATION_FLOW.md`

This file shows the exact setup order.
Do not improvise your own order.

---

### Step 3
Go to:

`01_SETUP/CUSTOM_INSTRUCTIONS.md`

Follow that file exactly.
This content belongs in ChatGPT Custom Instructions / Personalization settings.

Do not paste it into a normal chat.

---

### Step 4
Go to:

`01_SETUP/MEMORY_PROFILE.md`

Follow that file exactly.
This content is for persistent user preference / memory-style behavior.

If memory is unavailable, keep going.
The system will still work, but less strongly.

---

### Step 5
Go to:

`00_INSTALLER/INSTALLER_PROMPT.md`

Copy its full contents.

Then:
- open a new ChatGPT chat
- paste it as the first message

This turns ChatGPT into the installer and setup guide for the rest of the system.

---

### Step 6
In that installer chat, follow the instructions it gives you.

It should guide you through:
- checking whether setup was done correctly
- loading runtime files in the correct order
- preventing you from skipping required steps

---

### Step 7
When installation is complete, use:

`02_RUNTIME/CHAT_ACTIVATION.md`

This is the normal runtime activation entry for future chats.

---

### Step 8
Run:

`02_RUNTIME/RUNTIME_VALIDATOR.md`

Use it to confirm the system is behaving correctly.

If validation fails, use:

`02_RUNTIME/RUNTIME_RECOVERY.md`
