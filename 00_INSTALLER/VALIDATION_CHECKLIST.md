# VALIDATION CHECKLIST

Use this checklist after installation.

Do NOT assume the system is working.
Verify it.

---

## Setup Validation

Confirm:

- [ ] Custom Instructions are applied in ChatGPT settings
- [ ] Memory Profile is applied or acknowledged
- [ ] Installer was used (not skipped)

---

## Runtime Validation

Confirm:

- [ ] RUNTIME_FULL.md was pasted completely
- [ ] CHAT_ACTIVATION was used
- [ ] No setup files were pasted into runtime chat incorrectly

---

## Behavior Validation

Ask the system a real problem.

Check if:

- [ ] It asks clarifying questions instead of answering immediately
- [ ] It reduces options instead of expanding them
- [ ] It identifies real constraints or signals
- [ ] It does NOT validate everything you say
- [ ] It forces a direction instead of giving multiple equal options
- [ ] It gives a concrete next action
- [ ] It gives a gating condition (when to continue or pivot)

---

## Failure Indicators

System is NOT working correctly if:

- [ ] It gives generic advice
- [ ] It lists many ideas without narrowing
- [ ] It agrees with everything
- [ ] It avoids making decisions
- [ ] It produces vague outputs

If any of these happen:
→ system is NOT valid

---

## Recovery

If validation fails:

1. Re-check installation steps
2. Re-load runtime
3. Use:
   - 02_RUNTIME/RUNTIME_RECOVERY.md

---

## Final Condition

System is valid ONLY if:

- all required components are installed
- behavior matches system rules
- validation checklist passes

---

## Reminder

This system is strict by design.

If it feels different from normal ChatGPT,
it is working correctly.
